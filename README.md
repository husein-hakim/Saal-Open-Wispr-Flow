# Saal-Open-Wispr-Flow
An open-source, fully local alternative to Wispr Flow.

Current features:

- Local speech-to-text using whisper.cpp

- Local text rewriting using Qwen + llama.cpp

- Clipboard integration

- No cloud APIs

- No data leaves the device

Pipeline:

Microphone

→ Whisper

→ Qwen Rewrite

→ Clipboard

Roadmap:

- SwiftUI desktop app

- Global hotkey

- Real-time transcription

- VAD

- On-device optimization
